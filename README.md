Pixel Experience For Redmi K50
================

Pixel Experience is just a-click-away!
--------------------

**Notice: If you're using Windows, You may need Cygwin to run bash script**

Depends:
--------------
 - tar
 - openssl
 - pv
 - zstd
 - bash
 - fastboot

Install Pixel Experience in your device
-------------------

```
  bash ./install.sh
```
